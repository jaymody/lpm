# Lines Per Minute

A command-line typing tool made for programmers. Inspired by [wpm](https://github.com/cslarsen/wpm).

## Install
```
pip install git+https://github.com/jaymody/lpm.git
```

## Usage
Start the program with:
```
lpm
```
Use `lpm -h` for additional options.

## Dev Notes

**Install Project:**
```shell
pip install -e ".[dev]"
```

**Tests:**
```shell
pytest tests
```
